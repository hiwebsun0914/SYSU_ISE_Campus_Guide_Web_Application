# SYSU_ISE_Campus_Guide_Web_Application

中山大学 ISE 校园导览 Web 应用。

## 功能简介
- 提供校园导览入口与主要页面。
- 基于 Vite 构建的前端项目，支持本地开发与构建部署。

## 环境要求
- Node.js 18+（建议使用 LTS 版本）
- npm 9+（随 Node.js 提供）

## 本地开发
```bash
npm install
npm run dev
```

## 构建与预览
```bash
npm run build
npm run preview
```

## 目录结构
- `src/`：前端源代码
- `dist/`：构建产物（如已生成）
- `index.html`：入口页面

## 常用脚本
- `npm run dev`：启动本地开发服务器
- `npm run build`：构建生产版本
- `npm run preview`：本地预览构建产物
